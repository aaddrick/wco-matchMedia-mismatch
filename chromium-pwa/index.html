WCO matchMedia mismatch repro — Chromium PWA

Open this page over http://localhost in Chrome, then install it as an app (omnibox install icon, or … → Install …). The installed window declares display_override: ["window-controls-overlay"] so Chromium activates WCO and overlays the window controls onto the page.

The probe below queries the four standard WCO entry points. In an installed PWA window with WCO active, all four should agree (matchMedia returns true, the others are populated). In an ordinary browser tab, all four should agree the other way (everything false/0/empty).
